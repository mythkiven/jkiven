Description
立木征信用大数据消除信息不对称，提供更快、更精准的信用决策。依托数据挖掘分析和机器学习技术，对申请者提交的信息进行识别，并结合个人社交行为、运营商、身份属性、社保公积金及海量互联网信息。基于强大的数据点基础，3分钟即可完成评估。

目前在线可查询内容：
1.央行征信：个人征信报告（数据来自中国人民银行征信中心），快速查看信用卡记录、购房贷款记录以及其他贷款记录等；
2.社保/公积金：快速查询缴费单位、历史缴费记录、月缴额、余额等信息，目前已支持查询国内一线城市地区个人账号；
3.电商网购：直连京东\淘宝官网，个人网购信息一目了然；
4.运营商：数据覆盖三大运营商，速查个人通话记录、手机账单等信息；
5.学历学籍：涉及到全国高等教育学生信息，含学籍、学历、招生录取等信息；
6.职业社交：整合个人工作经历、人脉资源等职场社交信息形成职场画像分析报告；
7.失信被执行：全国法院失信被执行人名单信息查询；
8.信用卡账单邮箱：个人信用卡电子账单整合，生成账单报告；
9. 汽车保险：支持主流保险网站查询，通过个人授权，抓取车辆信息数据；
10. 网银流水：银行卡数据查询，查看个人交易流水。

另，支持API接口调用服务。访问官网：http://www.limuzhengxin.com/
